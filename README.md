# 📚 Projeto Vollun: Rede Social Literária Web

> **Status:** Em desenvolvimento 🚀 (Fase de Integração de APIs)

O **Vollun** é uma plataforma digital interativa projetada para conectar leitores e autores através de um ecossistema focado no compartilhamento de experiências literárias e organização de acervo pessoal.

## 🌟 Diferenciais do Projeto
- **Ecossistema Web:** Interface responsiva desenvolvida para navegadores modernos.
- **Social-First:** Engajamento através de curtidas, comentários e sistema de seguidores.
- **Inteligência:** Automação de metadados e busca de títulos via Google Books API.
- **Cloud Storage:** Gerenciamento seguro de arquivos PDF via Firebase Storage.

## 📂 Estrutura do Projeto

O ecossistema é dividido em duas frentes principais que trabalham em conjunto:

### 1. [Backend (Spring Boot)](./backend)
- **Tecnologias:** Java 21, Spring Boot 4.0.6, MySQL.
- **Responsabilidades:** API REST, segurança JWT e regras de negócio.

### 2. [Front-end (Angular)](./frontend)
- **Tecnologias:** Angular, JavaScript/TypeScript.
- **Responsabilidades:** Interface Single Page Application (SPA), consumo da API e visualização de PDFs.
