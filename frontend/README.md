# Vollun - Web Interface (Angular) 🖥️

Este repositório contém o front-end da plataforma **Vollun**. Desenvolvido com **Angular**, o objetivo é oferecer uma experiência de navegação rápida e intuitiva para quem busca gerenciar suas leituras e interagir com outros leitores.

## 🚀 Funcionalidades

- **Busca Dinâmica:** Componente de pesquisa que consome a integração com o Google Books.
- **Leitura Online:** Visualizador de PDF integrado para leitura direta no navegador.
- **Feed de Interações:** Espaço para acompanhar publicações, comentários e curtidas.
- **Perfil Personalizado:** Gestão de progresso de leitura e rede de seguidores.

## 🛠️ Tecnologias Utilizadas

- **Framework:** Angular 17+
- **Linguagem:** TypeScript / JavaScript
- **Estilização:** CSS3 / SCSS / Angular Material
- **Comunicação:** HttpClient (Consumo de API REST)

## ⚙️ Como Executar

1. **Instale as dependências:**
   ```bash
   npm install
   ```
2. **Iniciar serve**
   ```bash
   ng serve
   ```
