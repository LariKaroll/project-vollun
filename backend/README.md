# Vollun - Backend 📚

O **Vollun** é uma plataforma digital interativa focada no ecossistema literário, funcionando como uma rede social dinâmica que conecta leitores e autores. O projeto integra um **Gerenciador Literário e de Autoria (GLA)** com recursos sociais avançados.

## 🚀 Funcionalidades Atuais

- **Busca Externa:** Integração com a Google Books API para consulta de livros, autores e capas.
- **Segurança:** Estrutura baseada em Spring Security (em desenvolvimento).

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Java 17
- **Framework:** Spring Boot 4.0.0
- **Banco de Dados:** MySQL
- **APIs Externas:** Google Books API
- **Arquivos:** Firebase Storage (Planejado)
